Greybird for Pekwm

This is a port of the popular Greybird theme to Pekwm window manager.

Download Now

How to install

Just download the zip file and extract its content in the ~/.pekwm/theme/ directory

It has been tested in version 0.1.16 of pekwm.

Issues?

Just report them in the project page at Github.
